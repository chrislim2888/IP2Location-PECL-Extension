This IP2Location Demo BIN file is:

a) Incomplete - It contains data for IP address from 0.0.0.0 to 99.255.255.255 only.
b) Outdated - The data is not being updated regularly compare to commercial version.
c) Functional - The file format can be used directly with open-source API.

Please use online demo at http://www.ip2location.com/demo if you want to evaluate the latest results.
